# DrivingSchool
Вы - инструктор, который должен расставить конусы, чтобы ученики могли сдать зачёт по определённой задаче, используя автомобиль. Пользователь может брать клиентов и ставить их в зону ожидания, расставлять конусы на линии, чтобы клиенты могли сдать зачёт. Если конусы сбиты, пользователь должен их поднять и поставить на место, чтобы линия снова была готова для проезда. В игре также можно прокачивать спаунер или свою сумку, чтобы увеличить вместимость, или прокачать скорость персонажа, чтобы быстрее передвигаться. При накоплении определённой суммы можно обновить конусы на уровне или же перейти на новый уровень
